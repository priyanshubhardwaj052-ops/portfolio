# Priyanshu Bhardwaj | Developer & Data Specialist Portfolio

Welcome to my portfolio repository! This is a premium, modern, and fully responsive single-page portfolio website designed to showcase my qualifications, project portfolio, technical capabilities, and academic background.

Live Preview (Local Server Setup): `http://localhost:8000`

---

## 🚀 Key Features

*   **Premium Glassmorphic UI**: High-end dark theme design built with custom CSS variables, sleek radial background glows, and modern card backdrops.
*   **Interactive Particle Canvas**: Lightweight, performance-optimized HTML5 Canvas background particle system that dynamically connects nodes and responds to cursor movements.
*   **Dynamic Typing Animations**: Auto-typing text in the Hero section showcasing my core roles.
*   **Scroll-Spy Navigation**: Automatically highlights active sections in the glassmorphic navigation bar as the user scrolls.
*   **Filterable Projects Grid**: Visitors can filter between *Web Applications* and *Data & Analytics* projects instantly with interactive category tags.
*   **Serverless Contact Form**: Fully configured with **Web3Forms** for instant, AJAX-based email submission directly to my inbox without reloading the page.

## 💻 Local Development Setup

To run this website locally on your system:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/priyanshubhardwaj052-ops/portfolio.git
    cd portfolio
    ```

2.  **Start a Local Development Server**:
    *   Using Python:
        ```bash
        python -m http.server 8000
        ```

3.  **Open in Browser**:
    Visit [http://localhost:8000](http://localhost:8000) in your web browser.
